# Azure IaC + Observability Demo

Proyecto demostrativo para roles:

- Cloud Engineer
- DevOps Engineer
- Preparación AZ-104

## Arquitectura

- Azure App Service (F1)
- Application Insights
- Azure Monitor
- Alert Base
- Infraestructura como Código (Bicep)
- CI/CD con GitHub Actions

## Características

- Infra reproducible
- Observabilidad real
- Telemetría validada con KQL
- Deployment automatizado
